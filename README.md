Java System Tray Icons for Microsoft Windows
--------------------------------------------

This project contains the following system tray icons written in `Java` for
`Microsoft Windows XP and above`:
- `Stardate System Tray Icon` displays the current `stardate` and corresponding
date in other chronologies.
- `World Time System Tray Icon` displays the current date/time in multiple time
zones.


Installation
------------

You must have `Java 8` or better installed which is available for download from
[https://www.java.com](https://www.java.com).

If `Java 8` does not install:
- Try right click the `Java` installer and `Run as administrator`;
- Try an earlier archived version from
[https://www.oracle.com/java/technologies/javase/javase8-archive-downloads.html](https://www.oracle.com/java/technologies/javase/javase8-archive-downloads.html).

Each system tray icon's installer (`Setup.exe`) is located in the `Releases`
section.

If during the installation you see `Error opening file for writing`:
1. Abort and cancel the installation, then
1. Right-click the installer and select `Run as administrator`.


Stardate System Tray Icon
-------------------------

- Display the current `stardate`, in either the `classic` format or the newer
`2009 revised` format.
- Display the current date in other chronologies:
  - Coptic
  - Ethiopic
  - ISO (Gregorian)
  - Hijrah
  - Japanese
  - Julian
  - Minguo
  - Thai Buddhist


World Time System Tray Icon
---------------------------

- Display the current date/time in multiple time zones.
- Customisable date/time format.
- Customisable time zone names.
- Combine identical time zones.
- Specify a date/time for a given time zone and determine the corresponding
date/time in all other time zones.


Stardate Library
----------------

A `Java` library which converts to/from `stardate` and `Gregorian`.  Refer to
the `.jar` file in the `Releases` section and the corresponding `Javadoc`.
